# YouTube App UI Clone App

A UI clone of the YouTube mobile application.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npm run start
   ```

## TECH STACK USED

- React Native Expo
- Tailwind CSS
